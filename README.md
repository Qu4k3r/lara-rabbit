# Agilize Boilerplate

## 📋 Pré-requisitos
* docker
* docker-compose

## 🔩 Começando
Faça o fork desse repositório, pronto!

## 🔧 Instalação
Clone o projeto e entre em seu diretório

### Iniciando o container pela primeira vez
```sh
make build-and-serve
```

### Levante o container
```sh
make serve
```

## 🚀 Implantação

### Gere a chave da aplicação
```sh
make key-generate 
```
## ⚙️ Executando os testes
```sh
make all-unit-tests
```
